이미지 관련 태그

src 속성에 경로를 설정하여 이미지 파일을 불러온다.

alt 속성은 화면낭독기 기능 사용 시 화면 낭독기가 이미지를 읽을 수 없기 때문에
대체되는 텍스트가 필요하다.
혹은 사진의 경로가 잘못 되거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트의
용도로 사용 가능하다.

width와 height 속성

사진의 높이와 너비를 지정할 수 있다.

고정 크기 단위(px): 화면 사이즈에 상관없이 고정된 크기를 주고자 할 때 사용

가변 크기 단위(%): 화면 사이즈 혹은 기준이 되는 사이즈에 따라 크기가 변경될 때 사용

이미지 구역을 2개로 나누어 각각 링크 설정
